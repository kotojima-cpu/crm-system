# 画面仕様書

## 1. 画面一覧

### 1.1 WEB版（PC）

| # | 画面ID | 画面名 | URL | 概要 |
|---|--------|--------|-----|------|
| 1 | SCR-001 | ログイン | /login | メール・パスワード入力 |
| 2 | SCR-010 | 顧客一覧 | /customers | 検索・ソート・ページネーション付き一覧 |
| 3 | SCR-011 | 顧客登録 | /customers/new | 新規顧客フォーム |
| 4 | SCR-012 | 顧客詳細 | /customers/:id | 顧客情報＋紐付く契約一覧 |
| 5 | SCR-013 | 顧客編集 | /customers/:id/edit | 顧客情報の編集フォーム |
| 6 | SCR-020 | 契約登録 | /customers/:id/contracts/new | 新規リース契約フォーム |
| 7 | SCR-021 | 契約詳細 | /contracts/:id | 契約情報・残回数・ステータス表示 |
| 8 | SCR-022 | 契約編集 | /contracts/:id/edit | 契約情報の編集フォーム |
| 9 | SCR-030 | ユーザー管理 | /admin/users | 管理者のみ。ユーザーCRUD |

### 1.2 PWA版（スマホ）

PWA版はWEB版と同一URLだが、レスポンシブデザインでモバイル最適化表示となる。
編集・登録・削除ボタンはPWA版では非表示にする（ビューポート幅で判定、または専用フラグ）。

| # | 画面ID | 対応WEB画面 | PWAでの制約 |
|---|--------|-------------|------------|
| 1 | PWA-001 | SCR-001 ログイン | 制約なし |
| 2 | PWA-010 | SCR-010 顧客一覧 | 「新規登録」ボタン非表示 |
| 3 | PWA-012 | SCR-012 顧客詳細 | 「編集」「削除」ボタン非表示 |
| 4 | PWA-021 | SCR-021 契約詳細 | 「編集」「削除」ボタン非表示、残回数を目立つ位置に表示 |

## 3. 共通コンポーネント

### 3.1 ヘッダー（PC）

```
┌─────────────────────────────────────────────────────────────┐
│ [ロゴ]  顧客管理    [顧客一覧]  [ユーザー管理※]    [田中太郎 ▼] │
│                                                   [ログアウト] │
└─────────────────────────────────────────────────────────────┘
※ ユーザー管理は admin ロールのみ表示
```

### 3.2 ヘッダー（PWA/モバイル）

```
┌──────────────────────────┐
│ [≡]  顧客管理    [田中 ▼] │
└──────────────────────────┘
ハンバーガーメニュー:
  - 顧客一覧
  - ログアウト
```

### 3.3 ページネーション

```
[< 前へ] 1 2 3 ... 8 [次へ >]   全150件中 1-20件を表示
```

- 1ページあたり20件（デフォルト）
- PC: 数字ボタン表示、PWA: 「前へ」「次へ」のみ

### 3.4 検索バー

```
┌─────────────────────────────────┐
│ 🔍 会社名・担当者・電話番号で検索  │  [検索]
└─────────────────────────────────┘
```

- 入力後Enterキーまたは検索ボタンで検索実行
- API: `GET /api/customers?search=キーワード`

### 4.1 SCR-001: ログイン

**レイアウト**:
```
┌──────────────────────────────┐
│                              │
│      OA顧客管理システム        │
│                              │
│  メールアドレス               │
│  ┌──────────────────────┐    │
│  │                      │    │
│  └──────────────────────┘    │
│  パスワード                   │
│  ┌──────────────────────┐    │
│  │                      │    │
│  └──────────────────────┘    │
│                              │
│  [      ログイン      ]       │
│                              │
│  ※エラーメッセージ表示エリア    │
│                              │
└──────────────────────────────┘
```

**入力項目**:

| 項目 | 型 | 必須 | バリデーション |
|------|-----|------|---------------|
| メールアドレス | email | ○ | メール形式 |
| パスワード | password | ○ | 1文字以上 |

- ログイン成功 → 顧客一覧（/customers）へ遷移
- ログイン失敗 → 「メールアドレスまたはパスワードが正しくありません」を表示

### 4.8 SCR-022: 契約編集

SCR-020（契約登録）と同一レイアウト。

**差異**:
- タイトル: 「契約編集」
- 各フィールドに既存データがプリセット
- 顧客は変更不可（表示のみ）
- ステータスの手動変更が可能（cancelled への変更等）
- ボタン: 「更新」
- API: `PUT /api/contracts/:id`
- 成功時: 契約詳細へ遷移

### 4.9 SCR-030: ユーザー管理

**アクセス制限**: admin ロールのみ

**レイアウト**:
```
┌──────────────────────────────────────────────────────────┐
│ ユーザー管理                                  [新規登録]   │
├──────────────────────────────────────────────────────────┤
│ 名前       │ メールアドレス        │ ロール │ 状態  │      │
│───────────┼─────────────────────┼───────┼──────┼──────│
│ 管理者     │ admin@example.com   │ 管理者 │ 有効  │ [編集]│
│ 田中太郎   │ tanaka@example.com  │ 一般  │ 有効  │ [編集]│
│ 山田花子   │ yamada@example.com  │ 一般  │ 無効  │ [編集]│
└──────────────────────────────────────────────────────────┘
```

**機能**:
- ユーザー一覧表示
- 「新規登録」→ モーダルダイアログでユーザー作成フォーム
- 「編集」→ モーダルダイアログで編集フォーム（名前・メール・ロール・有効/無効）
- パスワード変更は編集モーダルからオプション（空欄の場合は変更しない）
- 自分自身の無効化は不可

**ユーザー登録/編集モーダル**:

| 項目 | 必須 | バリデーション |
|------|------|---------------|
| 名前 | ○ | 1〜100文字 |
| メールアドレス | ○ | メール形式、ユニーク |
| パスワード | ○（新規のみ） | 8文字以上 |
| ロール | ○ | 管理者 / 一般 |
| 有効/無効 | ○ | トグル |

## 5. レスポンシブ対応方針

| 項目 | PC（WEB版） | スマホ（PWA版） |
|------|------------|----------------|
| ブレークポイント | 768px 以上 | 768px 未満 |
| テーブル表示 | 通常テーブル | カード形式リスト |
| ナビゲーション | 水平メニュー | ハンバーガーメニュー |
| 登録/編集ボタン | 表示 | 非表示 |
| 検索バー | インライン | 上部固定 |
| ページネーション | 数字ボタン付き | 前へ/次へのみ |
