# OA機器販売 WEB顧客管理アプリ — 要件定義書

## 1. プロジェクト概要

OA機器（複合機・プリンター・シュレッダー等）の販売・リース契約を管理するWebアプリケーション。
営業担当者が顧客情報とリース契約を一元管理し、残回数の自動計算により契約満了時期を把握できるようにする。

### 1.1 基本方針

| 項目 | 方針 |
|------|------|
| 正本 | PC向けWEB版 |
| WEB版の役割 | 入力・編集・管理（フルCRUD操作） |
| スマホ対応 | PWA（Progressive Web App）として提供 |
| PWA版の役割 | 閲覧・検索中心（読み取り専用） |
| データ管理 | 共通DBで一元管理 |
| 残回数計算 | サーバー側で動的計算（`start_date` + `total_months` から算出） |

## 2. MVP機能一覧

### 2.1 WEB版（PC）

| # | 機能ID | 機能 | 概要 |
|---|--------|------|------|
| 1 | F-01 | ログイン認証 | メールアドレス＋パスワードによる認証 |
| 2 | F-02 | 顧客一覧 | 検索・ソート・ページネーション付き一覧表示 |
| 3 | F-03 | 顧客登録 | 新規顧客情報の登録フォーム |
| 4 | F-04 | 顧客編集 | 既存顧客情報の編集 |
| 5 | F-05 | 顧客詳細 | 顧客情報＋紐付く契約一覧の表示 |
| 6 | F-06 | 顧客検索 | 会社名・担当者名・電話番号等での絞り込み |
| 7 | F-07 | 契約登録 | 新規リース契約の登録（顧客に紐付け） |
| 8 | F-08 | 契約編集 | 既存契約情報の編集 |
| 9 | F-09 | ユーザー管理 | 管理者によるユーザーCRUD |

### 2.2 PWA版（スマホ）

| # | 機能ID | 機能 | 概要 |
|---|--------|------|------|
| 1 | P-01 | ログイン認証 | WEB版と同一の認証基盤を使用 |
| 2 | P-02 | 顧客一覧 | モバイル最適化された一覧表示 |
| 3 | P-03 | 顧客検索 | 会社名・電話番号等でのクイック検索 |
| 4 | P-04 | 顧客詳細 | 顧客情報＋契約一覧の閲覧 |
| 5 | P-05 | 契約閲覧 | 契約詳細の閲覧（編集不可） |
| 6 | P-06 | 残回数表示 | リース残回数のリアルタイム表示 |

> **PWA版の制約**: データの登録・編集・削除はWEB版でのみ行う。PWA版は閲覧と検索に特化する。

## 3. 仮定事項（未確認のデフォルト方針）

以下は未確認のため仮定を置いた事項。要件確定時に見直すこと。

| # | 項目 | MVP方針 | 備考 |
|---|------|---------|------|
| A-01 | ユーザーロール | 管理者（admin）・一般（general）の2種 | 閲覧制限なし（全顧客を全員が閲覧可） |
| A-02 | 顧客区分 | 法人のみ | 個人顧客対応はMVP後 |
| A-03 | 拠点管理 | 1顧客1住所 | 複数拠点はMVP後に拡張 |
| A-04 | 機器管理 | 契約に機種名・型番を文字列で保持 | 機器マスタ化はMVP後 |
| A-05 | リース会社 | 契約にリース会社名を文字列で保持 | マスタ化はMVP後 |
| A-06 | 残回数計算 | `start_date` + `total_months` からAPI応答時に動的計算 | DBに `remaining_months` カラムは持たない |
| A-07 | ステータス更新 | 日次バッチで `expiring_soon`（残3ヶ月以内）・`expired`（残0）に更新 | |
| A-08 | 満了通知 | 残3ヶ月以内の契約を一覧画面でハイライト表示 | メール通知はMVP後 |
| A-09 | ファイル添付 | MVP対象外 | |
| A-10 | CSV出力 | MVP対象外 | |
| A-11 | パスワードリセット | MVP対象外 | |
| A-12 | 活動履歴 | MVP対象外（訪問記録等） | |

## 4. 技術スタック

| レイヤー | 技術 |
|----------|------|
| フロントエンド | Next.js (React) + TypeScript |
| UIライブラリ | shadcn/ui + Tailwind CSS |
| バックエンド | Next.js API Routes (App Router) |
| ORM | Prisma |
| データベース | PostgreSQL |
| 認証 | NextAuth.js（Credentials Provider + JWT） |
| PWA | next-pwa（Service Worker + Web App Manifest） |
| バッチ処理 | Vercel Cron または node-cron |
| デプロイ | Vercel（または Docker） |
